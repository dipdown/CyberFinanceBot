# Installation Linux
1. Create Screen
   ```sh
   screen -Rd cfi
   ```
2. Clone Repository
   ```sh
   git clone https://github.com/dipdown/CyberFinanceBot/
   ```
3. Creating and Activating a Virtual Environment in Python
   ```sh
   python3 -m venv venv
   source venv/bin/activate
   ```
4. Add Query on (`key.txt`)
5. Running Program
   ```sh
   python3 main.py
   ```
